Inherits from NSObject
Declared in JDStatusBarNotification.h
JDStatusBarNotification.m

Overview

This class is a singletion which is used to present notifications on top of the status bar. To present a notification, use one of the given class methods.

addStyleNamed:prepare:

Adds a custom style, which than can be used in the presentation methods.

+ (NSString *)addStyleNamed:(NSString *)identifier prepare:(JDPrepareStyleBlock)prepareBlock

Parameters

identifier

The identifier, which will later be used to reference the configured style.

prepareBlock

A block, which has a JDStatusBarStyle instance as parameter. This instance can be modified to suit your needs. You need to return the modified style again.

Return Value

Returns the given identifier, so it can be directly used as styleName parameter.
